Comandos de Git



    Git init

    Inicia un nuevo repositorio

    Git clone

    Clona un repisotorio exitente desde la ubicacion remote

    Git Add .

    Agrega todos los archivos en el area de preparación

    Git Commit -m

    Crea un comentario con los cambios que se encuentran en el area de preparación

    Git status

    Muestra el estado actual del repositorio, incluyendo los archivos modificados

    Git push

    Envia los Commit locales al repositorio

    Git branch

    Muestra la rama actual en la que se encuentra en el repositorio

    Git pull

    Obtiene los ultimos cambios del repositorioremoto y los fusiona con tu rama local

    Git log

    Muestra u registro de los commits, mostrando el historial de cambios realizados en el repositorio

    Git remote add

    Conecta el repositorio local a un repositorio remoto especificado por la URL

    Git clean

    Elimina los archivos sin seguimineto de tu directorio de trabajo

    Git confit --global user.mane

    Nos genera el usuario el cual esta trabajando en dicho repositorio

    Git confit --global user.email

    Nos genera el correo el cual esta trabajando en dicho repositorio
