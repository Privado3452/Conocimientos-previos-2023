HTML

Estructura del Documento:

!DOCTYPE: Define el tipo de documento HTML.
html: Define el elemento raíz de un documento HTML.
head: Contiene metadatos sobre el documento.
title: Define el título del documento.
body: Contiene el contenido visible del documento.

Texto:

h1-h6: Define encabezados de nivel 1 al 6.
p: Define un párrafo.
span: Define un fragmento de texto en línea.
strong: Define texto importante, usualmente representado en negrita.
em: Define texto enfatizado, usualmente representado en cursiva.
abbr: Define una abreviatura o acrónimo.
cite: Define una referencia a una obra.
code: Define código de computadora.
pre: Define texto preformateado.
blockquote: Define una cita larga de otro autor.
q: Define una cita corta en línea.

Lista:

ul: Define una lista no ordenada.
ol: Define una lista ordenada.
li: Define un elemento de lista.

Vinculos e imagenes:

a: Define un enlace.
img: Define una imagen.

Tablas:

table: Define una tabla.
tr: Define una fila dentro de una tabla.
td: Define una celda dentro de una fila.
th: Define una celda de encabezado dentro de una fila.
thead: Define la cabecera de una tabla.
tbody: Define el cuerpo de una tabla.
tfoot: Define el pie de una tabla.

Formularios:

form: Define un formulario.
input: Define un campo de entrada.
textarea: Define un área de texto.
button: Define un botón.
select: Define un menú desplegable.
option: Define una opción dentro de un menú desplegable.
label: Define una etiqueta para un campo de entrada.

Multimedia:

audio: Define un elemento de audio.
video: Define un elemento de video.
source: Define múltiples recursos multimedia para elementos de audio y video.

Etiquetas semanticas:

header: Define la cabecera de un documento o una sección.
footer: Define el pie de página de un documento o una sección.
nav: Define un bloque de navegación.
section: Define una sección en un documento.
article: Define un contenido independiente y autónomo.
aside: Define contenido que está aparte del contenido principal, como barras laterales.
main: Define el contenido principal de un documento.
div: Define una división o contenedor genérico.
