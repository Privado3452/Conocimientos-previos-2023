CSS

Box Model:

width: Define el ancho de un elemento.
height: Define la altura de un elemento.
margin: Define los márgenes de un elemento.
padding: Define el espacio de relleno dentro de un elemento.
border: Define el borde de un elemento.
border-radius: Define el radio de borde para bordes redondeados.

Color y Fondo:

color: Define el color del texto.
background-color: Define el color de fondo de un elemento.
opacity: Define la opacidad de un elemento.

Tipografía:

font-family: Define la fuente de texto.
font-size: Define el tamaño de texto.
font-weight: Define el grosor de la fuente.
text-align: Define la alineación del texto.
text-decoration: Define la decoración del texto.
line-height: Define la altura de línea del texto.

Diseño de Caja:

display: Define el comportamiento de visualización de un elemento.
position: Define el método de posicionamiento de un elemento.
float: Define el método de flotación de un elemento.
clear: Define cómo los elementos deben comportarse en relación con los elementos flotantes.
z-index: Define el nivel de apilamiento de un elemento.

Animaciones y Transiciones:

animation: Define una animación.
transition: Define una transición entre dos estados.

Media Queries:

@media: Define reglas de estilo específicas para diferentes dispositivos y situaciones.

Pseudo-clases y Pseudo-elementos:

:hover: Define el estilo de un elemento cuando el usuario pasa el cursor sobre él.
:focus: Define el estilo de un elemento cuando recibe el foco.
::before: Inserta contenido antes del contenido de un elemento.
::after: Inserta contenido después del contenido de un elemento.

Unidades de Medida:

px: Píxeles, una unidad de medida absoluta.
%: Porcentaje, relativo al tamaño del elemento contenedor.
em: Relativo al tamaño de fuente del elemento.
rem: Relativo al tamaño de fuente del elemento raíz.
vh: Altura de la ventana, relativa al 1% de la altura de la ventana.
vw: Ancho de la ventana, relativo al 1% del ancho de la ventana.
